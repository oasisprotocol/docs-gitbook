# 账户列表

{% hint style="info" %}
假设你已经阅读了 [前提准备](prerequisites.md)环节和[设置](.../setup.md)环节。
{% endhint %}

要列出所有余额>0的抵押账户，请运行：

```bash
oasis-node stake list -a $ADDR
```
